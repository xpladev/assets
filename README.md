# Assets

## Layout

```
assets
│
└───cw20
│   │   pairs.dex.js # swap pairs.
│   │   tokens.js # cw20 tokens.
│
└───cw721
│   │   contracts.js # NFT collections.
│   │   marketplace.js # Open NFT marketplaces.
│
└───ibc
│   │   tokens.js # ibc tokens.
│
└───icon
│   │   extension # for extension.
│   │   game # for game.
│   │   svg # default icons.
│
└───vault
│   │    maintenance.js
│
└───chains.js # XPLA Vault chains.
│
└───contracts.js # XPLA Vault contracts.
│
└───extensions.js # XPLA Vault Wallet extension list.
```
